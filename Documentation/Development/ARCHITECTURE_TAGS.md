# Architecture Tags

Controlled terms for engineering records.

| Tag | Definition |
| --- | --- |
| `static-web-app` | Browser-only HTML/CSS/JavaScript app served from repository-root static files. |
| `sleeper-import` | Sleeper API player and trending-player import behavior. |
| `draft-state` | Draft picks, undo history, roster assignment, team names, and JSON state. |
| `player-prep` | Persistent per-player preparation fields including favorite, notes, tier, custom value, watch, and do-not-draft. |
| `live-auction-ui` | User interface optimized for fast live auction use and iPad landscape responsiveness. |
| `github-pages` | Repository-root static publishing through GitHub Pages. |
